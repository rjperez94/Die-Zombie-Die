## Controls
Key | Action
---|---
`W`, `Up` | Move Up
`S`, `Down` | Move Down
`A`, `Left` | Move Left
`D`, `Right` | Move Right
`E` | Interact
`Space` | Shoot
`Enter` | Select Option (Menu)
`Escape` | Pause Menu
